# TP01 – Productos y servicios que utilizan Machine Learning

## 1. Netflix

### Nombre del producto
Netflix

### Qué problema resuelve con ML
Netflix utiliza machine learning para recomendar películas y series personalizadas a cada usuario. Esto ayuda a los usuarios a encontrar contenido relevante sin tener que buscar entre miles de opciones.

### Qué tipo de ML usa
Aprendizaje supervisado y sistemas de recomendación.

### Por qué es ese tipo
El sistema aprende a partir del historial de visualización de los usuarios, utilizando datos previamente etiquetados sobre qué contenidos gustan o no a los usuarios.

### Qué datos necesita
- Historial de visualización
- Calificaciones o interacciones
- Tiempo de reproducción
- Preferencias de género
- Datos de usuarios similares

## 2. Spotify

### Nombre del producto
Spotify

### Qué problema resuelve con ML
Spotify usa machine learning para recomendar música personalizada y generar playlists automáticas como “Discover Weekly”.

### Qué tipo de ML usa
Aprendizaje supervisado y no supervisado.

### Por qué es ese tipo
El sistema analiza patrones en el comportamiento de los usuarios y agrupa canciones y gustos similares entre distintos usuarios.

### Qué datos necesita
- Canciones escuchadas
- Tiempo de escucha
- Canciones saltadas
- Likes o favoritos
- Datos de audio de las canciones (tempo, ritmo, etc.)


---

## 3. Google Maps

### Nombre del producto
Google Maps

### Qué problema resuelve con ML
Predice el tráfico y recomienda las rutas más rápidas para llegar a un destino.

### Qué tipo de ML usa
Aprendizaje supervisado.

### Por qué es ese tipo
Utiliza datos históricos de tráfico y velocidad de vehículos para entrenar modelos que predicen el estado del tráfico.

### Qué datos necesita
- Ubicación GPS de usuarios
- Velocidad de vehículos
- Datos históricos de tráfico
- Accidentes o eventos en la vía

## 4. Mercado Libre

### Nombre del producto
Mercado Libre

### Qué problema resuelve con ML
Utiliza machine learning para recomendar productos, detectar fraudes y mejorar las búsquedas dentro de la plataforma.

### Qué tipo de ML usa
Aprendizaje supervisado.

### Por qué es ese tipo
Los modelos se entrenan con datos históricos de compras, búsquedas y comportamientos de los usuarios.

### Qué datos necesita
- Historial de compras
- Búsquedas realizadas
- Productos vistos
- Datos de vendedores
- Información de transacciones


---

## 5. Tesla Autopilot

### Nombre del producto
Tesla Autopilot

### Qué problema resuelve con ML
Permite que los vehículos puedan conducir parcialmente de manera autónoma reconociendo objetos, carriles y señales de tránsito.

### Qué tipo de ML usa
Aprendizaje supervisado y aprendizaje por refuerzo.

### Por qué es ese tipo
El sistema aprende a partir de millones de datos recolectados por los vehículos y mejora su comportamiento en la conducción.

### Qué datos necesita
- Imágenes de cámaras del vehículo
- Sensores y radares
- Datos de conducción
- Mapas y entorno del vehículo
